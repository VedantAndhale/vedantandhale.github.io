There are Basically 4 steps in web scraping :
1. Send Request
2. Get Response
3. Select Data
4. Extract Data

![[image/How Web Scraping Works.png]]

When you send a request to a website URL, you receive a HTML Response. For Selecting data, you can use CSS selectors and XPath.
